# 🏎️ McLaren F1 Store – E-Commerce Web App

A sleek, modern **McLaren Formula 1–themed e-commerce store** built using **React**. This project showcases a stylish storefront experience inspired by motorsport branding, focusing on performance, UI design, and clean component structure.

🔗 **Live Demo:** [https://mclarenf1store-indol.vercel.app/](https://mclarenf1store-indol.vercel.app/)

---

## 📌 About the Project

This project is a **school project** created as part of an academic requirement. It focuses on building a full-stack e-commerce application using **React for the frontend** and **Node.js for the backend**, with a custom API deployed via Render.

It is designed with a **McLaren Racing aesthetic**, simulating a real online store where users can browse products, view featured items, and interact with a clean, responsive UI while communicating with a backend service.

The goal of this project is to:

* Practice **React development** and component structuring
* Build a **visually engaging UI/UX**
* Simulate an **online shopping experience**
* Deploy a real-world project using **Vercel**

---

## 🚀 Features

* 🏁 McLaren-themed landing page
* 🛍️ Product listing and featured products section
* ❤️ Wishlist functionality (UI-based)
* 🛒 Shopping cart interface (UI-based)
* 📱 Responsive design for different screen sizes
* ⚡ Fast performance using modern React practices

## ⚙️ Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/mclaren-f1-store.git

# Navigate into the project folder
cd mclaren-f1-store

# Install dependencies
npm install

# Start the development server
npm start
```
